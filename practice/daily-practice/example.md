# 일일 연습 예제

## 2024년 연습 기록

### Hello World 프로그램
간단한 "Hello, World!" 프로그램을 여러 언어로 작성해보기

#### JavaScript
```javascript
console.log("안녕하세요, 손승혜입니다!");
```

#### Python
```python
print("안녕하세요, 손승혜입니다!")
```

#### Java
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("안녕하세요, 손승혜입니다!");
    }
}
```

### 학습 내용
- 각 언어의 기본 출력 방법 학습
- 문법 차이점 비교
- 개발 환경 설정 연습
